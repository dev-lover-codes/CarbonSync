| Boundary | Description |
|----------|-------------|
| Client → API | Untrusted ID tokens from Firebase Client SDK are sent to `/api/login` |
| Middleware → App | Middleware injects verified session info into request headers/context |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-02-02 | Information Disclosure | Firebase Config | accept | Client-side Firebase config is public by design; secrets for Admin SDK are server-only. |
| T-02-SC | Tampering | npm installs | mitigate | slopcheck + blocking human checkpoint for [ASSUMED]/[SUS] |
</threat_model>
