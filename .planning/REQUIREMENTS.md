# Requirements: CarbonSync

**Defined:** 2026-06-08
**Core Value:** Providing an intuitive and engaging way for users to visualize their carbon footprint and take actionable steps to reduce it.

## v1 Requirements

### Infrastructure & Setup

- [ ] **INFRA-01**: Setup project with React + Vite
- [ ] **INFRA-02**: Initialize and configure Tailwind CSS with custom theme (primary, earth, sky, surface)
- [ ] **INFRA-03**: Configure Firebase App, Auth, and Firestore
- [ ] **INFRA-04**: Define Carbon Factors configuration for footprint calculation

### Authentication

- [ ] **AUTH-01**: User can authenticate (Sign up/Login) via Firebase Authentication

### Profiles

- [ ] **PROF-01**: Application creates and retrieves user profile documents
- [ ] **PROF-02**: User can update their profile information

### Activity Tracking

- [ ] **ACT-01**: User can log daily activities (transport, energy, food, shopping, waste)
- [ ] **ACT-02**: Application calculates CO2 emission impact for logged activities
- [ ] **ACT-03**: User can view their recent logged activities

### Goals

- [ ] **GOAL-01**: User can set carbon reduction goals
- [ ] **GOAL-02**: User can view and update their goals

### Insights & Leaderboard

- [ ] **INS-01**: User can view recent environmental insights/tips
- [ ] **LEAD-01**: User can view a leaderboard showing the top 10 users by total CO2 saved
