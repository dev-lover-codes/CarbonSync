  <files>src/lib/firestore-social.js</files>
  <action>
    Implement the logic for calculating progress towards collaborative group challenges:
    - Create `getGroupProgress(groupId)` function.
    - This function must:
      1. Fetch all user IDs in the group.
      2. Query the `activities` collection for all those user IDs within the current challenge timeframe.
      3. Sum the `impact` (CO2 saved) values.
      4. Return the aggregate total vs. the group's goal defined in the challenge document.
    - Optimize by using Firestore query `where('userId', 'in', memberIds)` if the group size is within Firestore limits (max 30 for 'in' query) or perform multiple queries/aggregation logic as needed.
  </action>
